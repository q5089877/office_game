# 性能優化與最終測試方案

## 概述
本方案旨在確保遊戲在所有目標裝置上都能提供流暢的性能體驗，並透過最終測試驗證所有功能的完整性和穩定性。

## 性能優化目標

### 1. 核心性能指標
- **幀率**: 穩定60fps (行動裝置), 60-144fps (桌面)
- **載入時間**: 首次內容繪製(FCP) < 1.5秒, 最大內容繪製(LCP) < 2.5秒
- **互動時間**: 首次輸入延遲(FID) < 100ms, 總阻塞時間(TBT) < 200ms
- **記憶體使用**: < 100MB (行動裝置), < 200MB (桌面)
- **網路使用**: 初始載入 < 2MB, 總資源 < 10MB

### 2. 裝置特定目標
- **低端手機**: 確保在4G網路和中等硬體上可玩
- **高端手機**: 利用高刷新率顯示器提供更流暢體驗
- **平板**: 優化大螢幕渲染和觸控互動
- **桌面**: 支援高解析度和多視窗

### 4. 記憶體優化

#### 物件池模式
```typescript
// 物件池用於頻繁創建/銷毀的物件
class ObjectPool<T> {
  private pool: T[] = [];
  private createFn: () => T;
  private resetFn: (obj: T) => void;

  constructor(createFn: () => T, resetFn: (obj: T) => void) {
    this.createFn = createFn;
    this.resetFn = resetFn;
  }

  acquire(): T {
    if (this.pool.length > 0) {
      return this.pool.pop()!;
    }
    return this.createFn();
  }

  release(obj: T): void {
    this.resetFn(obj);
    this.pool.push(obj);
  }

  clear(): void {
    this.pool = [];
  }
}

// 使用範例：卡片物件池
const cardPool = new ObjectPool<Card>(
  () => ({ id: '', name: '', description: '', type: CardType.SLACKING }),
  (card) => {
    card.id = '';
    card.name = '';
    card.description = '';
  }
);
```

#### 事件監聽器管理
```typescript
// 集中管理事件監聽器
class EventManager {
  private listeners: Map<string, Array<{ element: EventTarget; handler: EventListener }>> = new Map();

  add(element: EventTarget, event: string, handler: EventListener, options?: AddEventListenerOptions): void {
    element.addEventListener(event, handler, options);

    const key = `${event}-${Date.now()}`;
    if (!this.listeners.has(key)) {
      this.listeners.set(key, []);
    }
    this.listeners.get(key)!.push({ element, handler });
  }

  removeAll(): void {
    this.listeners.forEach((listeners, key) => {
      listeners.forEach(({ element, handler }) => {
        element.removeEventListener(key.split('-')[0], handler);
      });
    });
    this.listeners.clear();
  }
}

// 在組件中使用
const eventManager = new EventManager();

useEffect(() => {
  eventManager.add(window, 'resize', handleResize);
  eventManager.add(document, 'click', handleClick);

  return () => {
    eventManager.removeAll();
  };
}, []);
```

const performanceMonitor = {
  start() {
    this.metrics = {
      fps: [],
      memory: [],
      loadTimes: []
    };

    // 監控FPS
    this.fpsInterval = setInterval(() => {
      const fps = this.calculateFPS();
      this.metrics.fps.push(fps);

      if (fps < 50) {
        console.warn('低幀率警告:', fps);
      }
    }, 1000);

    // 監控記憶體
    if ('memory' in performance) {
      this.memoryInterval = setInterval(() => {
        const memory = (performance as any).memory;
        this.metrics.memory.push({
          usedJSHeapSize: memory.usedJSHeapSize,
          totalJSHeapSize: memory.totalJSHeapSize
        });
      }, 5000);
    }
  },

  calculateFPS() {
    // FPS計算邏輯
    return 60;
  },

  stop() {
    clearInterval(this.fpsInterval);
    clearInterval(this.memoryInterval);
    return this.metrics;
  }
};
