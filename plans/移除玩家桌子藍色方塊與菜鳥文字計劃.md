# 移除玩家桌子藍色方塊與菜鳥文字實施計劃

## 概述
本計劃旨在實現兩個主要功能：
1. 移除遊戲中玩家桌子上的藍色方塊和「菜鳥」文字
2. 實現玩家每次進入遊戲時隨機分配到空位子的功能

## 問題分析

### 當前狀態
在 `src/components/Canvas/OfficeCanvas.tsx` 第86-101行，玩家桌子的渲染包含：
1. **藍色方塊背景**：`fill={isPlayerDesk ? themeColors.primary[600] : "rgba(255,255,255,0.8)"}`
2. **「菜鳥」文字**：`text={isPlayerDesk ? "菜鳥" : desk.label}`

這個標籤組位於桌子本體的右下角（`x={5} y={45}`），用於標識桌子的所有者。

### 影響範圍
- 只影響玩家自己的桌子（`isPlayerDesk === true`）
- 其他NPC桌子的標籤保持不變
- 玩家角色頭上的名字標籤不受影響（仍顯示「菜鳥」）

## 實施方案

### 方案一：條件渲染（推薦）
修改 `OfficeCanvas.tsx`，當 `isPlayerDesk` 為 true 時，不渲染標籤組。

**修改前：**
```tsx
<Group x={5} y={45}>
  <Rect
    width={80} height={22}
    fill={isPlayerDesk ? themeColors.primary[600] : "rgba(255,255,255,0.8)"}
    cornerRadius={6}
  />
   <Text
    text={isPlayerDesk ? "菜鳥" : desk.label}
    fontSize={CANVAS_CONFIG.TEXT_SIZE.NPC.DESK_LABEL}
    fill={isPlayerDesk ? "#ffffff" : themeColors.secondary[400]}
    fontStyle="bold"
    width={80}
    align="center"
    y={5}
  />
</Group>
```

**修改後：**
```tsx
{!isPlayerDesk && (
  <Group x={5} y={45}>
    <Rect
      width={80} height={22}
      fill="rgba(255,255,255,0.8)"
      cornerRadius={6}
    />
    <Text
      text={desk.label}
      fontSize={CANVAS_CONFIG.TEXT_SIZE.NPC.DESK_LABEL}
      fill={themeColors.secondary[400]}
      fontStyle="bold"
      width={80}
      align="center"
      y={5}
    />
  </Group>
)}
```

### 方案二：空標籤
將玩家桌子的標籤設為空，但保留方塊（透明）。

**修改後：**
```tsx
<Group x={5} y={45}>
  {!isPlayerDesk && (
    <Rect
      width={80} height={22}
      fill="rgba(255,255,255,0.8)"
      cornerRadius={6}
    />
  )}
  <Text
    text={isPlayerDesk ? "" : desk.label}
    fontSize={CANVAS_CONFIG.TEXT_SIZE.NPC.DESK_LABEL}
    fill={isPlayerDesk ? "transparent" : themeColors.secondary[400]}
    fontStyle="bold"
    width={80}
    align="center"
    y={5}
  />
</Group>
```

### 方案三：完全移除組件
直接移除整個標籤組的渲染。

**修改後：**
```tsx
{/* 玩家桌子不顯示標籤 */}
{!isPlayerDesk && (
  <Group x={5} y={45}>
    <Rect
      width={80} height={22}
      fill="rgba(255,255,255,0.8)"
      cornerRadius={6}
    />
    <Text
      text={desk.label}
      fontSize={CANVAS_CONFIG.TEXT_SIZE.NPC.DESK_LABEL}
      fill={themeColors.secondary[400]}
      fontStyle="bold"
      width={80}
      align="center"
      y={5}
    />
  </Group>
)}
```

## 第二部分：玩家隨機初始位置

### 問題分析
目前玩家初始位置被硬編碼為 `(1, 1)`（在 `src/useGameEngine.ts` 第17行）。根據用戶需求，玩家每次進入遊戲時應隨機分配到空位子。

### 可用空位子
從 `src/constants.ts` 分析，有以下空位子可用：
1. `empty_l1` - (2, 1)
2. `empty_l2` - (2, 2)
3. `empty_m1` - (4, 1)
4. `empty_m2` - (5, 1)
5. `empty_m3` - (4, 2)
6. `empty_r1` - (7, 1)
7. `empty_r2` - (8, 1)

### 實施方案

#### 步驟1：創建輔助函數
在 `src/useGameEngine.ts` 中添加函數來獲取隨機空位子：

```typescript
const getRandomEmptyDesk = () => {
  const emptyDesks = OFFICE_LAYOUT.clusters.flatMap(c =>
    c.desks.filter(d => d.label === "" && d.owner === null)
  );
  if (emptyDesks.length === 0) {
    return { x: 1, y: 1 }; // 備用位置
  }
  const randomDesk = emptyDesks[Math.floor(Math.random() * emptyDesks.length)];
  return { x: randomDesk.x, y: randomDesk.y };
};
```

#### 步驟2：修改玩家初始化
修改 `src/useGameEngine.ts` 第17行：
```typescript
// 修改前
const player = new Character('player', '菜鳥', EntityType.PLAYER, undefined, 1, 1, Gender.MALE);

// 修改後
const randomDesk = getRandomEmptyDesk();
const player = new Character('player', '菜鳥', EntityType.PLAYER, undefined, randomDesk.x, randomDesk.y, Gender.MALE);
```

### 技術考量
1. **隨機性**：每次遊戲開始時隨機選擇
2. **空位子檢查**：確保選擇的是真正的空位子（`label === "" && owner === null`）
3. **備用方案**：如果沒有空位子，使用預設位置 (1, 1)
4. **一致性**：玩家桌子標籤移除後，空位子視覺上更自然

### 驗收標準
1. ✅ 玩家每次進入遊戲時位置隨機
2. ✅ 玩家只會被分配到空位子
3. ✅ 如果沒有空位子，使用預設位置
4. ✅ 遊戲其他功能不受影響
